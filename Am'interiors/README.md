# Am'interior

Am'interior is a small interior design company

# Requirements

For use Am'interior you have to start API.

### Getting started with the API

```bash
npm install
# or
yarn
```

```bash
$ cd api
```


### Run the app :

```bash
npm start
# or
yarn start
```
